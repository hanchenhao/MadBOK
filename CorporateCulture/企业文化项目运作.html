# 企业文化项目运作

企业文化要与生产经营深度融合，才能有作为、有价值、有意义。企业要结合实际情况，找准文化建设的对接点，着力点，通过实施企业文化项目化管理，推进企业文化示范点，创新企业专项文化，促进文化在业务管理领域全面落地。

# 建设量体裁衣

要使企业文化建设融入经营实践中，就要明确企业文化建设的目标和主体，结合企业的历史基因、业务性质、经营管理现状，采取适合自身实际情况的文化建设方法。在企业文化建设过程中，首先要明确自身目标，量身打造独具特色的企业文化。

通常情况下，国有企业和民营企业精神追求会有所不同，一般来说，国有企业会更加强调企业的社会责任和社会使命，为社会发展做贡献。民营企业则大多数着眼于战略成功、商业成功、市场竞争力提升等等。

国有企业要坚持党的领导、加强党的建设，因此，先进的党建指导思想和文化内涵是国有企业文化建设的理论和根基。国有企业在进行企业文化建设时，必须以当的政治文化为引领，将党建文化作为企业文化建设的重要一环。民营企业党建元素会弱一些，在大方向上会结合业务特性，参考全球最佳标杆公司的实践和经营哲学，进行因地制宜的创新，推动企业文化与经营实践深度融合。

在明确企业的精神追求、文化建设理论根基等目标后，不管是什么类型的企业，其建设主体都离不开客户、员工、股东、社会责任这些维度，这些维度作为建设关注的焦点，提炼相应维度的文化理念和行为要求。

# 推进示范建设

为了有序推进企业文化建设，要把项目管理的思想、策略手段和方法运用到企业文化建设的执行工作过程中，以具体的项目带动文化落地。利用项目激发企业活力，并逐步推进文化的层层落实。企业文化建设可以将一个大项目分解成若干个小项目，并监控器持续落实的过程。

很多企业是按照项目计划与启动-分类推进-中期评估-结项评审的形式来做全过程管理。启动阶段明确项目需求，深入研究业务特色制定实施方案，梳理各阶段的成果和产出。中期评估时，让各单位、各部门梳理项目的阶段性情况，回顾项目完成情况，分析项目进展过程中存在的问题和不足，并制定相关优化对策。结项时，梳理已经取得的项目实施成果，形成结项报告，评选出优秀成果进行表彰，以达到激励效果。

做好企业文化建设，关键在于执行，企业要根据各阶段工作计划，合理调配内部资源，确保文化建设能够获得有力的支持。组织要提供如下三类必要保障：

  • 人员保障。要做好人员的能力培养，能够促使每一个项目都分工明确、责任清晰，使得项目组的每一个成员都能获得直接快速的人力支持。
  • 组织保障。可以制定专门落实企业文化建设的规章制度，用制度的形式为项目化建设提供支持，保障企业文化建设的运营。
  • 思想保障。要了解员工的思想动态，及时研究集中反馈的重点、难点、热点问题，充分调动员工的积极性，以减少项目推进的阻碍。

为了打造企业文化的新局面，可以利用标杆的力量，建设企业文化示范点，在企业内创建一定的建设示范样板，形成以点带面的示范效应。在创建企业文化示范点时，要把我如下要点提炼特色文化内涵，同时梳理示范点建设思路，最终明确示范点建设方法推进示范点建设。
