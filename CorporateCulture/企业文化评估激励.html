# 企业文化评估激励

要做好企业文化建设，就要建立完善的文化评估机制，落实对企业文化建设情况的评估，推动企业文化的循环改进。要建立有助于企业文化落地的激励机制，为员工创造良好的工作环境，才能保障企业文化的落地。

# 建立企业文化评估机制

企业文化评估机制是企业文化建设与落地工作的重要推动力，是对企业文化建设的过程与成果的检验体系。企业文化建设是一项长期、系统、复杂的工程，很难一步到位，需要在过程中不断改进。可以借助PDCA循环来推动企业文化建设的循环改进：

  • Plan（计划）阶段：要做好调查研究，分析企业文化现状，找出企业在文化建设过程中存在的问题，研究人员要尽可能的深入现场，做到逐个问题、逐个细节的排查，找到问题的缘由，再此基础上制定解决措施，提出相应的计划方案，并进行效果预测。计划方案要包括工作细节的排查、部门职责的划分、岗位职责的细化、工作的时间节点、规章制度以及激励方案等等。
  • Do（实施）阶段：对Plan阶段的计划内容进行处理和解决，进行具体实施。该阶段强调全员参与，作为管理者要以身作则，带头示范，组织开展各种类型的活动，充分调动员工参与文化建设的热情。
  • Check（检查）阶段：对计划执行的过程进行跟踪反馈，并评估计划执行的效果，一旦发生偏差，马上进行调整，对业务流程、员工行为进行优化和规范。
  • Action（改进）阶段：对监督和检查过程发现的问题有针对性的进行处理和改进，并将检查结果与激励管理联系起来，引导员工做出有利于企业文化发展的行为。本阶段未解决的问题，可以导入下一个PDCA循环，进行持续改进。

为了有效解决企业文化建设中存在的突出问题，系统提升企业文化建设的工作水平，企业需要建立完善的文化评估机制，对企业文化建设各个阶段取得的成果进行科学的评估。要想建立起一套完善的、能够有效推动文化落地的评估机制，首先要做两件事，一个是做好角色划分，一个是明确评估内容。

评估内容将会涵盖一下几个方面：

  1. 对文化管理尽责情况进行评估
  2. 对文化形象传播效果进行评估
  3. 对管理者个人的尽责效果进行评估
  4. 对制度文化建设效果进行评估

# 落实企业文化建设评估

为了落实对企业文化建设情况的评估，企业要对可以利用哪些评估指标有所了解，强化文化建设过程中的督导检查力度，并运用好评估结果。从宏观上讲，企业文化的评估可分为过程性评估和结果性评估两类，只有综合采用两种评估形式，才能全面把握企业文化的落地情况。

  • 过程性评估：主要检查原先计划工作的实施进展情况，企业文化落地规划就是过程性评估的主要依据，据此可以检验各项活动和措施的组织实施情况。主要涉及培训宣贯、物化展示、内部宣传、外部传播、领导关注、管理示范、典型示范、制度配合、仪式活动、行为标准、行为环境等方面。
  • 结果性评估：倾向于检验文化落实的综合效果，其主要包括企业内部员工对企业文化的认知度、认同度、对企业的满意度、敬业度。员工行为、企业制度与企业文化的匹配度，客户对企业文化的认知度、认同度，客户对企业的忠诚度，以及企业的社会美誉度等。

在具体的企业文化建设考核评估的实践中，企业可以根据实际情况，设计合适的评估指标，综合运用过程性和结果性评估，全方位检验企业文化的落地情况。为了及时了解企业文化的落实情况，必须强化建设情况的督导检查工作，可以引入包括“自查、督查、检查”在内的稽查管理办法。也可以强化工作汇报机制，定期了解员工思想动态来构建一个企业文化持续闭环的管理系统。

# 建立企业文化激励机制

合理有效的激励机制是促使企业文化落地的重要手段之一。要建立企业文化落地激励机制，就要基于员工需求匹配激励方式，善用奖惩结合的形式，将员工与企业的利益绑在一起。

如果想正向有效的引导员工行为，就必须准确把握和满足员工的真正需求，只有基于员工的需求来设计激励方式，才能发挥真正有效的激励作用。在企业管理活动中，我们可以将马斯洛的需求层次理论运用在员工激励与行为引导上。

在建设激励机制时，要把我奖励和惩罚的要点，使二者结合，强化员工的文化认同感。要用好奖励，适当惩罚，胡萝卜加大棒混合激励。想让激励机制发挥作用，必须确保员工与企业利益是一致的，或者说要有效地将员工与企业利益绑在一起，企业要最大限度的将那些对企业做出过贡献的员工纳入事业共同体中来，当员工的利益切实与企业的利益关联起来时，员工才会拼尽全力。
