# 智慧城市推进策略

智慧城市在具体建设中涉及主体众多，用户需求模糊多变，各类数据形式多样，信息相互关系错综复杂，分析模型个性化，建设项目众多，这些因素都会导致在具体的建设过程中容易偏离预定目标，建设效果差强人意，如何科学的开展智慧城市建设成为摆在人们面前的难题。

要解决这个问题就需要明确两点，其分别是做什么和什么时候做。做什么解决建设内容，其要点是需要根据城市的特点和需求，均衡地确定需要建设哪些项目以及建设规模如何，梳理清楚项目之间的匹配关系，并确保这些项目整合之后能够达到预期的总体目标。什么时候做是解决建设步骤，其要点是根据每个阶段的目标，确定项目的完成次序，从而确保没有超前建设、过渡建设和滞后建设的项目。

# 智慧城市发展的几个阶段

智慧城市发展涉及经济、社会、环境等各个方面，必须是一个循序渐进、不断发展的过程，在此过程中，智慧城市呈现出如下几个阶段性特点：

  • 起步阶段：此阶段应制定长远发展目标，编制智慧城市顶层规划，出台建设重要保障和配套制度，统一和明确智慧城市建设的重要作用和意义，为下一阶段快速发展做好思想、智力、财力和物力的准备。此时，城市的信息系统以单个、总想系统为主，以提升办公效率为目标，系统被动接受人的指令，数据处理、科学计算作为信息系统的主要功能，数据决策能力弱。
  • 发展阶段：此阶段重点开展基础性、框架性、公共性的项目建设，搭建智慧城市总体架构，围绕着城市重点发展需求，开展典型、示范性智慧应用建设工作。建立数据规范和交换标准，对城市的事件、部件、物件等统一编码，形成”城市语言“。在数据方面，可以通过初步挖掘之后的决策信息，形成一定的知识模板库，对已有经验进行积累和沉淀。基础性的项目主要包括基础通信网络建设、智慧城市公共信息平台、城市基础数据库、云计算平台、一网通办等等。智慧应用可以围绕着民声、政务、经济、环境等方面展开建设。
  • 优化阶段：此阶段信息分析和挖掘利用得到进一步强化，使信息化不仅是城市管理和运行的辅助工具，同时能够成为城市竞争力的重要组成部分，成为继人力、财力、物力之外重要的城市资源。智慧城市处理大量视频流数据、图片数据、文本数据、社交行为数据等非结构化数据，通过数据挖掘手段，向城市管理者提供关联性很强的决策方案。信息系统方面，以纵横联合为主，集成化开发，确保城市环境友好，人流、物流的顺畅、人力资源的高效培养，城市经济的可持续发展。
  • 成熟阶段：此阶段各项重要系统基本建设完备，智慧城市建设重点主要集中在如何让信息在城市各个需求主体间流动，在信息系统和数据分析决策方面，以信息系统的综合集成、整合利用为主，处理结构复杂、类型众多的数据，城市的管理、运行和民生均能通过信息分析和挖掘达到最佳状态，并对城市未来发展提供科学依据。

# 智慧城市建设原则

明确了建设目标后，需要遵循如下原则，才能保证建设工作的顺利开展：

  1. 政府统筹、市场运作：强化政府在规划引领、统筹协调、优化环境、政策扶持、应用示范等反面的主导作用，建立统筹领导体系，统一顶层设计，形成信息化工程间的协同运行。发挥市场机制在资源配置中的基础性与先导性作用，构建政府、企业、市民等多方参与、充满活力的发展格局。
  2. 整合资源、协同共享：以信息共享、互联互通为重点，突破区划、部门、行业界限和体制性障碍，充分整合信息基础设施和城市公共信息资源，推进跨部门、跨领域的信息化协同，提高城市整体运行效率和管理服务水平，提升全社会信息资源的利用水平。
  3. 立足民生、推动产业：面向公共生产、生活、学习的多元化需求，提供随时、随地、随需、低成本的信息与应用服务，使社会和市民享受到城市精细化、智慧化管理带来的便利和实惠。以智慧城市建设拉动智慧产业的需求，推动信息通信技术与产业的深度融合，促进高端制造业和服务业形成和发展。
  4. 循序渐进、重点突破：充分认识智慧城市建设的系统性和复杂性，从政府组织管理体系、城市信息化系统顶层设计、城市信息通信基础设施建设以及成熟应用示范工程等多维度切入、循序渐进，切实推动城市智慧化协同发展。结合实际，突出应用，讲求实效，找准产业升级、民生诉求的突破口先试先行。

# 智慧城市的实施路径

智慧城市建设需要城市的方方面面参与，良好的建设策略能够将发挥出最大限度的建设效果，如下是智慧城市建设的路径参考：

  1. 制定顶层设计和总体规划：结合城市发展战略，充分考虑城市主题对智慧城市的建设需求，以可持续发展理念为指导，抓住城市演进本质与城市发展关键资源要素进行顶层设计，全面、多维、立体地分析智慧城市视角下的城市要素体系、运行管理体系、公共服务体系、技术支撑体系、法律保障体系、目标评估体系等架构与内在逻辑。
  2. 重点工程和示范应用：智慧城市是一个涉及各领域、跨机构、跨部门的复杂系统工程，需要聚焦城市发展和重点任务，遵循“科学性、基础性、紧迫性、操作性、创新性、效益性”等标准，选取战略意义大、示范带动性强、现实需求迫切、受益面广的重点工程和示范项目，集中优势资源，在短期内取得成效，形成系统化、高效化的推进模式。可行的做法是选择一批重点项目作为智慧城市第一阶段优先建设项目，为后续建设打下坚实的基础。
  3. 延伸应用，分阶段推广：智慧城市建设需要分阶段进行，并针对不同的情况及时做出调整，不能因为智慧城市建设影响到当地发展和居民生活，根据阶段重点建设效果，不断积累和总结成功经验做法，以点及面，以局部示范带动全面应用，以资源整合促进融合提升，逐步扩大应用领域范围，有计划、有步骤地进行推广。
  4. 建设评估和规范制定：建立信息化工程项目的绩效评估和考核机制，落实各类建设管理规范的制定，将智慧城市建设项目纳入重点考核目标范畴，实现对项目建设进度的实时跟踪监督，定期发布考评结果，保障城市建设项目切实执行、稳步推进、取得实效。

智慧城市在实施的过程中可能会出现很多问题，推动实施路径有效落地，需要兼顾考虑过去资源、现实问题和未来规划，其中顶层设计至关重要，其主要体现在如下几个方面：

  • 回归城市建设本质。明确蓝图和计划，坚持战略引领、问题导向、需求驱动，明晰智慧城市建设蓝图和实施计划，掌握未来城市发展的蓝景、时间表以及预算投入，明确业务发展目标，指引业务流程优化。
  • 避免信息孤岛和重复建设。在中国条块分割的行政体系下，智慧城市实施人员陷入各自为政、信息孤岛等信息化建设的老问题。建设者应该避免设计局限化，要做好政策规划与技术实施两个层面的顶层设计，要保障信息化建设具有开放、融合和服务的特地啊，使之可以实现充分的资源整合和共享。
  • 构建协调机制。分界出具体的建设任务，小布快跑，实现渐进式的项目推进，分阶段促成智慧城市建设愿景，明确城市系统内部各部门的业务范围以及相应的责任，加强部门之间的分工合作，避免管理分散，解决项目建设、业务协同与资源共享的问题。
