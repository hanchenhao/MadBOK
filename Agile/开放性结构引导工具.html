# 开放性结构引导工具

什么是 "Liberating Structures"？Liberating Structures是由30多种互动模式组成的集合，这些模式可以在群体中展开、丰富和深化互动，由Keith McCandless和Henri Lipmanowicz所收集的，具有强大的复杂性科学根基的解放结构，所有的Liberating Structures都实现了五个设计元素和十个原则 。这意味着Liberating Structures并不是一套僵化的、静态的形式。新的结构一直在开发，现有的结构也在被修改、调整或删除。有了 "Liberating Structures"，就可以通过调动群体和个人的互动方式，开始引导整个组织变革。

开放性结构引导工具是一种用来让每个人都可以参与到共创中的互动方法，它可以促进团队积极活跃的参与活动，从而使真正包容和释放每个人的潜能成为可能。如果你是面对引导团队开展活动手忙脚乱、不知所措，可以试着用一下这些"Liberating Structures"工具。

作为一个高效的、成功的Scrum Master需要各种各样的技能、知识和经验。这种多样性体现在Scrum Master的立场上，Scrum Master应该充当老师、障碍清除者、引导师、教练、导师和变革者。一切的一切，都是以服务型领导（servant-leadership）为基本心态和态度。Liberating Structures 使Scrum Master能够根据这些立场来履行他们的角色。

# 1-2-4-ALL

1-2-4-ALL这个工具可以让每个人参与到讨论中来。它使很多开放的、有生产力的对话能够得以展开，想法和解决方案能够被迅速筛选。

使用目的：

  • 让每个人都参与寻找答案
  • 避免过度帮助和过度控制的恶性循环
  • 创造安全的表达空间，减少权利差异
  • 表达无声的对话，扩展输入的多样性
  • 建立共识或共同理解

适用场景：

  • 在演讲或演示后，需要丰富的反馈时，可以使用该方法
  • 可以利用此方法设计周会
  • 适用于宣布会议主题后开始的自发谈话
  • 适用于为解决某个问题而进行的小组公开讨论
  • 代替领导者告诉人们该怎么想，怎么做

适用时长：10-20分钟

分组方式：先是个体，然后是两人，然后是四人，最后是所有人

活动步骤：

  • 引导者提出问题后，由个人进行默默的自我反思。问题可以是：您认为在这次挑战上取得进展的机会是什么？您将如何处理这种情况？您建议采取什么想法或行动？
  • 两人互相说明自己的想法，在自我反思的基础上达成共识，形成统一的想法。
  • 在四人组中分享并发展彼此的想法，形成一个有代表性的统一想法。
  • 每个四人小组派一个代表与大家分享一个重要想法。

小贴士：

  • 在配对谈话前，需要引导大家进行安静的自我反思，请大家默默记下自己的想法。
  • 坚持精准的时间，必要时才再做一轮。
  • 将分享的想法限制在三到四个范围内。
  • 不要重复分享其他组已经提出过的见解。
  • 将小组讨论的内容与分组产生的想法分开并加以保护。

# 欣赏式访谈

欣赏式访谈可以在短时间内，帮助任何规模的小组找到其成功的关键条件。通过寻找当前有效的方法和发现成功的关键因素，去开展积极的行动。团队成员在分享他们成功故事时会充满活力。

使用目的：

  • 以积极的口吻开始，激发建设性的能量。
  • 制定创新战略，推进工作进展。
  • 捕捉和传播成功经验的隐性知识。
  • 为团队揭示成功路径。
  • 激发同伴间的学习，激发小组创新。

适用时长：60分钟

分组方式：先2人一组，然后4人一组，最好让不熟悉的人配对

活动步骤：

  • 明确一个主题。（3分钟）
  • 学员两人一组，轮流采访，讲述一个成功的故事，关注成功的原因。（15-20分钟）
  • 在四人小组中，每个人讲述配对伙伴的故事，请参与者倾听其成功的关键因素。并将小组的见解记录下来。（15分钟）
  • 收集各小组的见解，让全体成员都能看到，并对情况进行总结。（10-15分钟）
  • 提问：我们是如何促进成功的，成功的关键因素是什么？接下来我们可以做什么？（10分钟）

小贴士：

  • 先给大家讲个故事，或者玩个游戏来破冰。
  • 请大家给自己小伙伴的故事起一个标题。
  • 让每个人都能看到这些成功的故事。
  • 让故事开始丰富细化，为团队愿景带来生命力。

# 携手共画

通过携手共画可以拦住人们获取隐藏的知识，难以用语言表达的东西，反而可以画出来。这种方式提供了一种全新的表达途径，刺激参与者的想法。

使用目的：

  • 获得口述无法获得的见解或理解。
  • 挖掘带来创新的知识来源。
  • 发展团队对复杂状况的共同理解

适用时长：40分钟

分组方式：先个人单独练习画符号，然后个人制作图画初稿，由小组内解读图画后，进行全员汇报总结。

活动步骤：

  • 主持人介绍携手共画符号的含义：圆形=整体，矩形=支持，三角形=目标，螺旋=变化，星星=关系。（5分钟）
  • 请大家练习画这些符号（5分钟）
  • 请每个人结合符号，单独不加文字的创作一个故事初稿，讲述工作中的挑战或日常的工作历程。（10分钟）
  • 请每个人创作第二稿，通过戏剧化的符号大小、位置和颜色来完善故事。（10分钟）
  • 请参与者邀请另一个人或他们的小组解释每一张画，其中创作该画的人不能说话。（5分钟）
  • 让全组的人一起讨论：这些画揭示了什么？（5分钟）

小贴士：

  • 提醒学员绘画本身不是目标，不需要关注绘画技巧
  • 帮助参与者画出任何必要的东西
  • 用相机记录参与者的绘画过程
  • 记录绘画时参与者的情绪反应

# 三人行咨询

通过三人行咨询这一工具，看帮助人们深入了解他们所面临的问题，并释放出解决问题的智慧。在快速循环的“咨询”中，每个人寻求帮助，并能立即得到另外两人的建议。三人行咨询服务于任何希望从他人那里获得帮助的人。

使用目的：

  • 提高寻求帮助的技能。
  • 学会明确提出问题和挑战。
  • 完善倾听和咨询技能。
  • 通过相互支持建立小组内的信任。
  • 构建自我组织的能力。

适用时长：30分钟

分组方式：3人一组，尽量把不同背景和视角的人放在一个小组里。

活动步骤：

  • 请每一位参与者思考一下自己做客户时所要寻求咨询的问题。（1分钟）
  • 各小组请第一位客户分享自己的问题。（1分钟）
  • 两位咨询师向客户提出澄清提问。（1-2分钟）
  • 客户转身，背对两位咨询师
  • 咨询师一起讨论想法、建议以及辅导意见。（4-5分钟）
  • 客户转过身来分享刚才获得的宝贵想法是什么。（1-2分钟）
  • 各组切换到下一个人，重复上述步骤。

小贴士：

  • 请与会者组成不同角色/职能的小组
  • 让参与者试着留意提供支持的方式。告诉参与者在保持同理心的基础上尝试探索。
  • 保持安全空间，使用能激发客户自我理解或自我修正的提问
  • 告诉客户尝试并专注于自我反思，问：“究竟发生了什么？我是如何体会正在发生的事情的？”

# 25/10集体智慧

25/10集体智慧可以快速生成并筛选团队中有利的可行想法。它让我们在30分钟内，帮助一大群人产生并整理想法，以便采取行动

使用目的：

  • 培养团队快速挖掘自己的想法。
  • 获得更有可能持久的结果。
  • 激发不同观点之间的协同作用，同时建立一致性。
  • 鼓励创新思考，拿出切实可行的第一步。
  • 创造一个能够冒出好想法的环境。

适用时长：30分钟

分组方式：

  • 每个人想出一个大胆的想法，并写在索引卡上。
  • 每个人站着传递卡片。
  • 结对交流想法。
  • 每个人对自己手上的卡片进行打分。
  • 全组分享最终的最高分和想法。

活动步骤：

  • 主持人向大家解释一下过程。过程是：让每个参与者在索引卡上写下自己的想法。然后让大家自由走动，卡片从一个人传给另一个人，让大家快速浏览，当时间到了的时候，人们停止传递卡片，结对交流手中卡片的想法，然后参与者分别对卡片上的想法和步骤进行打分，分数从1分到5分写在卡片背面。这样共进行五轮计分，在第五轮结束时，参与者将他们所持最后一张卡片的背后5个评分相加，找出得分排名前列的创意，与全组共享。（3分钟）
  • 用一张样本索引卡演示一次交换和评分的互动，以澄清自由走动过程中的期望，不要大声朗读卡片，只在人与人之间传递卡片，使每个人手中都只有一张卡片。（2分钟）
  • 请每位学员在自己的卡片上写上一个大胆的想法和解决方案的第一步。（5分钟）
  • 进行5轮3分钟的交换和评分，中间包含自由走动的时间。（15分钟）
  • 请参与者将手中的卡片背面分数相加，与全组人员一起寻找最高分的想法。（5分钟）
  • 提问：“25/10吸引你注意的是什么？”（2分钟）

小贴士：

  • 如果评分不稳定，可以利用平均值来进行计算。
  • 请小组选择一个大胆的想法和行动步骤，并对其进行修改，使其表达的更有说服力。
  • 制定一个认真有趣的评分标准，利润1分=我不喜欢，5分=让我兴奋不已等等。
  • 为了不让参与者看到前几轮的得分，可以在卡片的背面贴上一张便签。

# 最小规格

最小规格可以明确为了实现目标的绝对“必须做”和“不必做”的工具。通过规定最低数量的简单规则，就必须遵循最小规格。最小规格是一种赋权约束，它帮助我们消除那些非必要的规则，帮我们解构阻碍创新的“最大规格”。

使用目的：

  • 评估并决定什么是成功的必要条件。
  • 为新的可能性开放出空间。
  • 将资源和精力集中或转移到重要的地方。
  • 在快速发展的市场中简化策略

适用时长：30-50分钟

分组方式：先从个人开始，然后扩展至4至7人一组，最后全组分享

活动步骤：

  • 生成所有必做和不做的活动清单，先是每个人一分钟独立思考，然后在小组内巩固和扩展五分钟，在段时间内尽可能完整的列出清单。（6分钟）
  • 每个小组根据目的声明测试其最大规格列表中的每一个规格，将如果范围规格扔能实现目的的内容从列表中移除。（15分钟）
  • 必要的时候做第二轮。（15分钟）
  • 各小组比较，精简到最小清单。（15分钟）

小贴士：

  • 从完整的注意事项开始，让尽可能多的相关方参与进来。
  • 权利追求最小规格。
  • 如果小组遇到困难，可以一起复盘，以阐明弥补，确保其归结为真正重要的事情。

# 共识-确定性矩阵

共识-确定性矩阵将探讨的挑战问题分为简单、相对复杂、极其复杂和混乱几个类型。它可以帮助个人或团队避免频繁的出现错误。

使用目的：

  • 通过将挑战与方法相匹配来减少浪费。
  • 确定有没有团队本地经验可以帮助解决问题。
  • 让每个人都看到组织中人们面临的挑战。
  • 通过识别不匹配的地方来减少人们在关键挑战上没有取得进展的挫败感。
  • 在组织中跨职能和级别之间分享观点。

适用时长：45分钟

所需空间及材料：

  • 4-6人一组的椅子。
  • 开放式的墙壁，墙上贴着大幅的矩阵插图。
  • 为每个人准备一张空白矩阵图。
  • 为每个人准备便利贴和记号笔。

分组方式：先个人单独进行初步评估，然后4-6人一组进行确认，最后进行整组讨论

活动步骤：

  • 要求参与者单独列出挑战清单。（5分钟）
  • 参与者将列出的挑战放入个人矩阵中。5分钟
  • 要求参与者结对讨论。5分钟
  • 邀请结对者和形成4-6人小组一起讨论，找到一致和不一致的地方，以及不匹配的地方。（10分钟）
  • 邀请每一个人发表他们的挑战，张贴在墙上的大矩阵中。（5分钟）
  • 要求参与者组成小组，退一步反思：“我们看到了什么模式？是否有什么需要解决的不匹配的问题？”（5分钟）
  • 邀请全组分享意见并决定下一步行动计划。（10分钟）

小贴士：

  • 明确挑战的类型是什么，以及要包含在内的活动。
  • 从个人开始，然后结对圆桌交流。
  • 避免对个人选择的活动地点进行评价。
  • 通过询问：“这个挑战示范具有多种动态？它是如何既简单又复杂”来探索多个领域的项目。

# 关键-不确定

关键-不确定能够在一系列看似合理但不可预测的未来中开发策略。可以帮助一个多元化的团体快速测试当前策略的可行性，并建立其快速应对未来挑战的能力。

使用目的：

  • 通过暴露假设和不确定因素来测试当前策略的可行性。
  • 提高每个人快速适应和弹性吸收干扰的能力。
  • 在稳健和对冲策略方面区分优先次序。
  • 建立组织信心，扩大战略选择范围。

适用时长：100分钟

所需空间及材料：

  • 四组桌子和对应的椅子
  • 每组用纸、便签、白板

分组方式：将人数分为四个独立小组，制定四个场景和相关策略

活动步骤：

  • 说明步骤的顺序。（2分钟）
  • 请参与者列出他们所面临的不确定因素清单，问他们：“在你工作的环境中，有哪些因素是无法预测或控制的？”（5分钟）
  • 确定关键因素的优先级，问他们：“哪些因素威胁到我们取得成功？”（10分钟）
  • 根据小组的历史经验，选择最关键的因素（X）和最不确定因素（Y）。（5分钟）
  • 在白板纸上画一个有X轴和Y轴的坐标图，标注X值和Y值，并将其分为四个象限。（5分钟）
  • 请四个小组分别为其象限取一个名字，并写出这个象限中可能出现的业务场景。（10分钟）
  • 四个小组简单分享自己的场景。（8分钟）
  • 每个小组集思广益，提出三种策略，帮助小组所在描述的场景中成功落地。（10分钟）
  • 四个小组简单分享各自的策略。（8分钟）
  • 各组交换讨论结果，以确定哪些策略是稳健的（可以在多个象限成功的策略），哪些是对冲的（只能在一种场景下成功，但可以在保护我们避免受到可能的风险）。（10分钟）
  • 四个小组分享汇报，全组对自己要执行的第一步工作进行决策。（10分钟）

小贴士：

  • 在头脑风暴中讨论不确定因素时，回忆一下出错的预测和让大家措手不及的事件。
  • 有趣的设计场景，例如，将它们变成关于未来情况的新闻报道。
  • 用便签纸组合想法。
  • 鼓励所有人说出自己的见解。

# W³反思法

W³反思法是指What、So What、Now What？，可以组织团队一起回顾迄今为止的工作进展，并决定需要进行哪些调整。它可以帮助团队对共同的经历进行反思，以建立一种方式，加强理解和促进协调行动，同时避免冲突。

使用目的：

  • 对人们如何形成不同的观点、想法以及行动和决定的理由有共同的理解。
  • 确保从分享经验中产生学习。
  • 避免不断重复同样的错误。
  • 避免因不清楚事实或对事实的解释而形成争论。
  • 共同学习、建立信任、减少恐惧。

适用时长：45分钟

所需空间及材料：

  • 供5-7人小组坐的椅子，配套小桌
  • 每组用纸、便签、白板

分组方式：先个人单独成组，再5-7人一组，最后扩展至集体

活动步骤：

  • 分组并描述活动规则。
  • 第一阶段讨论：What。个人单独思考1分钟，想想“工作中发生了什么，我们注意到了什么，哪些事实或观察比较突出。”然后在小组内进行集中讨论。（3-8分钟）
  • 小组中的突出事实与其他组进行分享，并收集反馈。（2-3分钟）
  • 第二阶段讨论：So What。大家单独思考1分钟，想想“为什么这个很重要？这个过程出现了什么模式或结论？我们可以提出什么假设？”然后在小组进行集中讨论。（3-8分钟）
  • 将小组的突出模式、假设和结论与全组分享，并收集反馈。（2-5分钟）
  • 第三阶段讨论：Now What。大家单独思考1分钟，想想“先按照怎么办？哪些行动有意义？”然后在小组中进行集中讨论。（3-8分钟）
  • 总结行动并与全组分享讨论，请大家补充见解。（2-10分钟）

小贴士：

  • 与小组一起检查，以澄清每个问题的答案。
  • 当小组分享时，每次只收集一个重要答案。
  • 不要试图收集重复答案，寻找独特答案。
  • 欣赏并认可坦诚的反馈。

# 听到-看到-尊重

听到-看到-尊重可以练习更深入的倾听和与同事的共鸣。通过这个引导方式，可以培养他人共情能力。共情以包容性的方式消除行动障碍，通过伙伴关系创造力量，解决互动之间的心结和矛盾。

使用目的：

  • 揭示人们不被倾听、不被看到或不被尊重的经历有多普遍。
  • 提高小组成员之间的倾听、调整和共情能力。
  • 帮助管理者认识到倾听比试图解决问题更有效。

适用时长：35分钟

所需空间及材料：面对面拜访椅子，不需要桌子

分组方式：先成对讲故事，再四人一组反思发生的事情

活动步骤：

  • 介绍本次活动的目的，倾听过程中不提供任何建议，不做出任何判断。（3分钟）
  • 每人每次有7分钟的时间分享一个关于不被倾听、不被看见或者不被尊重的故事。（15分钟）
  • 伙伴互相分享倾听和讲故事的经验：“讲我的故事是什么感觉？听你的故事是什么感觉？”（5分钟）
  • 在四人组中讨论：“故事中透露出哪些模式？这些模式对我们来说有多重要？”（5分钟）
  • 作为整个小组来思考一下问题：“用什么行动可以解决大家讨论的问题？”（5分钟）

小贴士：

  • 分享第一个跳出脑海的故事，这个故事往往是最好的。
  • 为了保障安全，可以让参与者不选择最痛苦的事情，要保护隐私。
  • 不要判断是非对错。
