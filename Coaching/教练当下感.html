# 教练当下感

IFC要求教练有能力保持清醒觉察并与客户建立自然的关系，采用一种开放的、灵活的、自信的工作方式，具体如下：

  1. 在教练过程中是全身心投入和灵活的，能够活在当下。
  2. 利用自己的直觉并相信内在洞察力，有体察本能反应。
  3. 愿意探索未知并冒险。
  4. 能够看到帮助客户的多种途径，并能够选择出最有效的一种来来帮助客户。
  5. 能够有效的运用幽默创造轻松和能量。
  6. 应对强烈情绪时，能够自我调节不受客户的情绪影响。

这项能力的基础是伙伴关系。这种伙伴关系由教练对教练对象的整体（情绪、能量、智慧和学识）深度的觉察和包容而得到加强。这种伙伴关系是双向的，有时候最适合教练对象的做法是要求他们在教练过程中发表观点，有时候对他们最有效的做法是教练介入并采取行动去转变能量或在某个特定方向提出挑战。

“当下感”这个概念是多维度的，它是一种感觉和一种体验，也是一系列存在、行为和联结方式的结果，带来与某人同在当下的体验。具有当下感能够支持教练对象更好的支持思考过程，创造一个舒适的环境。具有“当下感”的教练，能够越来越深入的使被教练对象有自己的思想和领域，而不是强迫他们接受教练的思维方式。

教练对象如何思考，如何创造或展现生命中的事物，他们最佳的学习方式，他们的情绪，他们如何联结他人，他们的价值观以及信念构成了教练的“世界观”。这个世界观决定了他们理解世界和日常生活的方式。例如，他们把当挑战当做一个问题还是一个邀请？当他们想要生活中有所改变时，会制定一个线性计划并一步一步执行吗？这一切构成了他们现在是谁，他们如何在自己的生活中表现。

要把教练对象当做一个有经验、视角、感情和价值观的个体，教练不能一味地解决问题，而是要对话题背后的人感到好奇。要好奇的点不只是他们需要做出什么改变，还应该包括他们如何认识这个话题的，他们的认识产生了什么效果，他们现在是谁，最终想成为什么样子。下面的例子是常见的教练干预：

  • 我经常听到你说到自由，听起来它对你很重要是么？
  • 他总是做出这种事，这给你传递了什么信息？
  • 回想一下学习这项技能对你起的作用是什么？
  • 这种行为让你有什么感觉？
  • 当你产生这种情绪的时候，你的行为会受到什么影响？
  • 假如你成了领导者，你会成为怎样的一个人？
  • 如果你这样做的话，它会给你的表现带来什么不同？

有很多种方式去表现善解人意和积极回应，对于教练来说，要对被教练对象“无条件正向关注”，不管他们感受到什么、不管他们怎么思考、不管他们想分享什么，教练都要接收它，以宽容和不评判的方式做出反应，然后用自信和稳定的状态去回应。当被教练对象有强烈的情绪时，教练要保持平静并好奇的了解对方需要什么，而不是试图修复这种情绪或者让不快乐的人高兴起来。

强烈的情绪是一种能量的运动，当教练对象体验到强烈情绪时，预示着他有某种改变或表明改变的需要，作为教练需要有能力与它同在，并有效地驾驭它。“有效地”不意味着使教练对象回到更舒适的状态，而是指去帮助教练对象去发现是什么导致这种情绪，以及那对其可能意味着什么。人们往往哭诉后才意识到某种处境使他们不幸福，意识到问题后才能下定决心解决问题。教练的工作就是注意到这些情绪，保持平静和好奇，陪着教练对象一起去探索未知。

为了保持好奇心，教练就要学会放弃自己的准则，并善于观察和乐于接受教练对象的标准。每个人看待世界的视角都不一样，视角不同决定每个人的标准不同，教练不能坚持认为自己是正确或者有用的。教练可以在谈话中邀请被教练对象分享自己的直觉或智慧，在结束的时候也可以互相讨论其中的收获，教练的好奇可以帮助他们对自身、思维和行动方式有一种新的认识。

# 核心能力要点

  1. 着眼于对方整体，关注他们是谁、他们想要什么、他们能够成为谁。倾听他们在生活中如何表现自己，这些表现对生活现状产生了什么影响。
  2. 关注教练对象的能力，表现在他们的声音、语调、节奏和身体等方面。
  3. 学会放下自己的标准，理解教练对象的准则。
  4. 关注伙伴关系，共同设计教练过程，帮助他们表达学习收获，听到多种选择时，询问他要考虑哪一个选项。
