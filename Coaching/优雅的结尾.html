# 优雅的结尾

设计行动、计划目标和管理进展汇聚在一起可以为谈话创造出一个“优雅的结尾”。这一组能力共同为对话创造一个水到渠成的结尾，它能够帮助对象全面评价面向目标取得的进展，同时总结学习整体的收获，在总结的过程中可以选定行动计划并对其有着清晰的理解。

# 设计行动

设计行动是关于谈话之外的推进行动，要求教练有能力与客户创造机会不断地学习，并采取新的行动来有效地带来约定地教练成果，ICF具体定义如下：

  1. 拓展客户思路，协助客户确定行动方案去展示、练习或深化新的学习收获。
  2. 帮助客户聚焦于约定的教练目标，并探索相关问题以及机遇。
  3. 促使客户去探索可选择的想法和方案，评价各个选项并做出相关规定。
  4. 推动积极尝试和自我发现，由此将收获应用于客户的工作和生活。
  5. 挑战客户的前提和假设视角，激发新的想法并找到新行动的可能性。
  6. 支持或提出与客户目标一致的视角，以不带偏好的方式，促使客户去深入考虑。
  7. 在教练会谈中帮助客户“现在就做”，提供即时的支持。
  8. 鼓励全力以赴的挑战，支持舒适的学习节奏。

设计行动要在教练谈话中推进即刻启动，教练应该与教练对象建立某种水平的伙伴关系去设计行动计划，并且承诺的行动应该目的明确地朝着目标去推进。

# 计划目标

计划目标是指帮助教练对象面向未来设计目标去实现长期、可持续的改变。教练要有能力与客户生成并保持一个有效的教练计划，ICF的具体定义如下：

  1. 合并所有搜索信息，针对客户关注的问题确立教练计划和成长目标。
  2. 可以建立一个成果是可实现的、可衡量的、具体且有时限的计划。
  3. 能够根据教练会谈的进展对计划作出必要的调整。
  4. 帮助客户发现并获得不同的学习资源，例如书籍和其他专业人士。
  5. 发现并指出对客户很重要的早期成功。

这项能力要求教练帮助被教练对象不仅对明确提出的目标进行探寻，还要求教练始终关注教练对象，确保适时地检查和调整对应计划，以促进教练对象能够随时保持正确的方向。其中对成果和成功的关注可以是具体的、可衡量的收获，也可以是有关于教练对象的领悟、觉察和自信等内容。

# 管理进展

教练有责任持续关注教练对象所说他们想要的东西，并且支持他们朝那个方向探索、成长和行动。ICF的具体定义如下：

  1. 明确的要求客户采取行动去推动他们朝着确定的目标前进。
  2. 进行后续行动，针对上次会谈承诺的行动询问进展情况。
  3. 有效地与客户处理或回顾会谈期间获得的信息。
  4. 使客户在会谈期间保持正轨，始终关注教练计划、目标成果、约定的行动方式以及对未来会谈的话题。
  5. 聚焦于教练计划，但是同时会根据教练会谈的进展和方向来调整行为方式和行动。
  6. 能够来回穿梭于客户的现实和愿景之间。
  7. 促进客户自律，使客户承担起责任。
  8. 培养客户能力，使他们能够做出决定，解决关键问题并且发展自己。

这项能力要求帮助教练对象展望前方，做出计划和安排，考虑各种阻碍计划的事项，并进行相应的调整。教练要保持着对被教练对象无条件的正向关注，要相信他们有能力取得成功，有必要的时候才以正向的方式向他们指出承诺与计划之间的出入。
