# 职业标准和职业道德

IFC对于教练的道德行为准则的定义主要分为一般行为、利益冲突、职业行为、保密隐私、持续提升几个方面，具体如下：

  1. 在所有互动包括教练培训、教练指导、以及教练督导等活动中，遵守ICF道德准则之要求。
  2. 承诺与教练、培训师、或者教练导师之间采取恰当的行动，一旦注意到任何违背或者可能破坏道德准则的情况，会联系ICF进行处理，无论涉及到的是我还是他人。
  3. 与他人包括组织、雇员、发起人、教练以及其他有需要的人进行沟通，让他们了解此道德准则所规定的责任。
  4. 在职业活动中，戒除出现不合法的歧视，包括年龄、种族、性别、族群、性取向、宗教、国籍或残疾等。
  5. 做出关于我作为教练所提供的服务、教练职业、或者ICF的口头或者书面的真实、准确的声明。
  6. 准确地说明我的教练资格、专业知识、经验、培训、证书和ICF的认证资质。
  7. 承认并尊重他人的努力和贡献，只会对自己的资料宣称所有权。我理解如果违反这项要求可能会导致第三方追究我的法律赔偿责任。
  8. 时刻努力发现那些可能会对我的教练效果或我的职业教练关系产生削弱、抵触或妨碍作用的个人问题。无论何时，如果客观情况需要，我会迅速寻求有关专业人士的协助，并确定要采取的行动，包括决定是否有必要暂停或者终止我的教练关系。
  9. 认识到道德准则适用于我与教练客户、被教练者、学生、被指导者以及被督导者之间的关系。
  10. 按照公认的科学标准，在适用的题目范围内，运用我的能力、本着诚实的态度进行研究并公布结果。我的研究将会得到所涉及到的相关人员的同意和认可，并且将以一种保护参与者不会遭受任何伤害的方式进行。所有研究行为将符合所在国家的所有适用法律。
  11. 以一种保密的、安全的、保护隐私的、并遵守任何适用的法律法规的方式，维护、存贮、处置在教练过程中产生的任何记录，包括电子文档及沟通信息。
  12. 只按照ICF许可的方式和程度使用ICF成员的联系信息（如电邮地址，电话号码等）。
  13. 设法察觉任何利益冲突以及潜在的利益冲突，当出现冲突时，公开披露这些冲突，并主动提出离开。
  14. 明确内部教练的角色，设定边界，与利益相关者一起检查在教练角色以及其他角色职能之间可能出现的利益冲突。
  15. 向客户及发起人披露由于推荐客户我可能从第三方得到的所有预期报酬，或者因为得到客户而支付的酬金。
  16. 给予每段教练/客户关系以平等的尊重，一视同仁，无论报酬的形式如何。
  17. 合乎道德规范地向客户、潜在客户或者发起人讲述我认为真实的关于教练过程或者我作为教练的潜在价值。
  18. 在第一次见面前或见面时，仔细地解释并努力确保我所教练的客户及发起人理解教练的性质、保密的性质和界限、财务安排、以及教练协议的其他条款。
  19. 在开始教练关系前，与我的客户及发起人之间约定清晰的教练服务协议，并尊重这份协议。这份协议会包括相关各方的角色、责任以及权利。
  20. 在与客户或发起人之间进行身体上或者其他方面的互动时，我会承担起保持觉察的责任，设定清晰、恰当、符合文化习惯的界限。
  21. 避免与我现在的客户、发起人、学生、被指导者或者被督导者产生任何性关系或者浪漫关系。此外，我会充分意识到包括我的员工或助理在内的各方之间产生性亲密关系的可能性，并采取适当行动解决问题，或者取消雇佣关系，以提供安全的整体环境。
  22. 尊重客户按照协议规定在教练过程中随时终止教练关系的权利。我会留意客户从教练关系中获得的价值产生变化的迹象。
  23. 鼓励客户或者发起人转为由其他教练或者资源来提供服务，如果我认为这样对他们会更好的话。当我认为必要或者合适时，我会建议客户寻求其他职业人士的服务。
  24. 对所有客户及发起人的信息进行最严格的保密，除非法律要求公布。
  25. 关于如何在教练、客户以及发起人之间交换教练信息，约定明确的协议。
  26. 作为教练、教练导师、教练督导或者培训师，与客户及发起人、学生、被指导者、或被督导者之间，就不能维持保密的情况（例如，下达法院指令或传票的非法活动；对自己或他人即将产生的或者可能的危险；等等）约定明确的协议，并确保客户和发起人、学生、被指导者、或被督导者在知情的情况下，自愿地以书面形式同意这些保密的限度。当我有理由相信上述某种情况适用时，我可能需要通知有关当局。
  27. 要求所有与我共同工作、一起支持我的客户的人员遵守ICF道德准则第4节“保密 /隐私”之第26条的规定，以及此道德准则中其他任何可能适用的章节的规定。
  28. 承诺继续和持续提升我的专业技能。

ICF对于教练的评估指标主要是判断是否表现出违反道德规范，以及是否有脱离教练角色而进入其他角色，比如顾问、教师、导师或者咨询师等。我认为这些要求的实质是保护被教练者。如果作为教练，在不具备专业知识以及基本行为道德准则的情况下，误入某些领域执行教练过程，可能会对被教练者造成伤害。比如，当和抑郁症患者对话时，如果没有经过专业的心理学训练，就不该贸然介入。因此，这项能力的关键是要限制自己的工作内容，如果是做教练就要时刻问自己：“我这是在进行教练吗？”

# 违反道德规范的行为

作为教练，我们可能会发现我们需要在教练过程探讨情感，而且有时候我们可能会选择去探讨从过去获得的学习收获。当这两件事同时发生并且教练陷入情感的过去时，谈话可能就可能会集中于了解过去的强烈情感，如果过去对于现在的他们造成了很大的障碍，那么教练可能就会陷入治疗模式。大部分的教练其实是不具备心里治疗专业能力的，所以进入这种模式首先会违反职业道德行为，其次也是对被教练者不负责的表现。为了避免这种情况，要清楚教练技术不仅是一个解决问题的手段，它更多是发现和释放潜能的工具，我们在沟通的过程中很自然的聚焦在想要发生什么，以及如何做好前进的准备，才会阻止我们落入想要了解和解决过去的陷阱。

如果教练对象咋诉说一件本应引起警惕的事情，而教练没有意识到这个问题的严重性，那也是一种丧失规范的行为。例如，教练对象想要解决工作上的问题，但是他对教练说连续几个月感觉到很抑郁，看不到希望并且经常忍不住哭泣，但是不知道是什么原因。如果教练听到这一切却没建议对方考虑心理治疗，那么其实这就是一种失职。诊断抑郁症不是教练的工作，当一个问题超出教练工作界限时，教练需要有能力进行识别。应对这种情况，教练要改进自己的直觉和价值观，教练必须明白如何在他们想要帮助及他们认为应该有能力帮助的地方和实际上他们有资格帮助的地方划清界限。如果对话超过了教练的能力范围，就需要考虑支持被教练者寻求其他帮助。

# 脱离教练角色的行为

在教练术语中，开放性问题是指能够让人更广泛回答的问题。一般来说，这些问题都会包含“什么”、“如何”、“谁”这类的词。很多教练觉得开放性的提问不是难题，但是他们可能会无意识的通过提问引导或影响被教练对象朝着自己偏好的解决方案前进，尽管这些提问看起来是开放的，但这是一种不信任被教练对象的表现。

“我很担心年底被裁员，我觉得工作做得不够充分，我想要通过教练来帮助我制定应对计划。”“你会如何通过人力资源部门或者就业组织讨论裁员后的再就业计划？”这个例子就是教练已经在自己的头脑中想出了一个解决方案，并且引导着对方向自己的解决方案前进。这就是脱离教练行为的表现。2023年1月9日

提供建议也是脱离教练角色行为的表现，但是很多人感觉无法控制自己不提建议。一个合格的教练，要充分信任被教练对象能够做出自己最好的思考。教练主要是集中询问和解释，谈话的内容主要是现在和将来。

# 核心能力要点

  • 确保我们是在教练。
  • 尊重被教练对象说的话，以及他们的思考方式。
  • 帮助被教练对象从过去吸取教训，而不是帮他们医治过去。
  • 必要的时候，结束教练过程。
