# 达成教练合约

IFC对于这项能力的定义是要求能够了解在具体教练活动中需要什么，并且就教练过程和关系与其去年在客户和新客户达成一致。具体体现是：

  1. 理解并与客户有效地讨论教练关系的指导原则和具体因素，如费用、时间安排、后勤保障等。
  2. 就什么是在教练关系中恰当的、什么是不恰当的、要提供什么，不要提供什么，以及客户与教练的责任达成一致。
  3. 判断教练方法和潜在客户的需求之间是否有良好的匹配。

这项能力的关键是由教练创造合作关系的深度。这种合作关系包括并落实如何进行教练会谈以及会谈中将探讨的话题。教练必须去探讨被教练对象想要什么，并且帮助他们真正说清楚教练目标成果。达成教练合约其实没有什么固定的做法，它可能会以不同的次序，出现在不同阶段的谈话中。但是不论以何种方式完成，它都要尽量实现以下目标：

  • 同被教练对象想要的会谈结果保持一致。
  • 要用足够的时间去理解被教练对象真正想要在谈话中聚焦的问题。
  • 与被教练对象双方都清楚为什么想要这个结果。
  • 同被教练对象形成伙伴关系。
  • 拥有一个会谈中进行核查的基准，以便能确定我们是否在正确的轨道上，或者需要再必要的时候进行调整。

假如教练会谈成果在会谈中发生了改变，那么重新确认新出现的目标成功并且按新的约定进行教练是必要的。但是导致这种原因的本质可能是因为教练在进行第一阶段谈话时过于草率，因此字教练合约阶段一定不要匆忙，教练合约为整个谈话奠定基础，所以要明智的投入时间。

# 帮助客户确定目标及衡量标准

教练要询问会谈的话题和目标，并帮助教练对象确切的阐述目标是什么。如果接受一个模糊的目标，那后面的教练过程会变得非常困难。教练应该能够清晰准确的把目标成功回讲给被教练对象确认，许多教练一心想帮忙，以至于他们最终探讨起教练对象可以采取哪些行动去抓住这次机会，这时候可能很多被教练者会卡主，因为他们并没有准备好讨论这些更深层次的内容。

教练假设自己听明白被教练者的目标这件事，很容易 限制谈话的力量和深度，很多时候教练对象可能并不确切的知道他们说话的含义，因此需要教练挑战他们给出更加明确的说明，以便他们对会谈需要去往哪里有更好的察觉。比如，话题是“探讨职业规划”或者“探讨数字化转型”，有些教练感觉他们理解这些术语，并且很高兴在这方面进行教练，因此错过了“职业规划”和“数字化转型”对被教练对象意味着什么。所以，应该问清楚关于“职业规划”和“数字化转型”他想重点探讨什么，或者他们目前遇到的挑战是什么。

接下来说一下实现目标的衡量标准，很多被教练者会把这个标准说成像是“成功的衡量标准”的一番话，这时候就要对齐进行进一步询问，如果不去寻找真正成功的衡量标准或者接受模糊的标准，那么教练过程会偏离应有的方向。例如，如果要讨论一个关于“和上级拥有良好关系”的话题，就要询问：“在这次会谈结束时，是想要感受到已经和上级建立起良好关系了吗？”这类的探寻。进一步讨论目标成果及衡量标准，能让对象真正的识别具体目标以及目标成功的标准。

如下是常见的教练干预对话：

  • “今天你想要在哪方面进行教练？”
  • “是什么让你着急做这件事呢？”
  • “跟我谈谈，这件事对你来说意味着什么？”
  • “这个话题你重点关注的是什么？”
  • “今天讨论的话题，哪个方面会给你带来最大的收益？”
  • “今天会谈理想的目标成果是什么？”
  • “你将如何确定在会谈结束的时候达到了这一点？”
  • “你将如何知道已经实现了今天的目标成果？”

# 探讨重要或有意义的成果

教练有些时候由于过度忙碌，很少有时间停下来反思并意识到被教练对象真正想要实现什么，所以他们往往急于找到解决问题的方法，在解决事务性问题的过程中，教练还要围绕这意义和重要性进行一些探讨。当询问重要性和意义时，被教练对象会感觉到自己被聆听。他们将感受到更多的理解和尊重，同时将有可能真正放松的进行会谈。

当教对象不清楚或不了解真正的意义或他们追求的实质是什么时，他们就很难发挥思维的创造力。探寻目标的重要性和结果的意义，将促使他们能更好的利用直觉，并做出更好的选择。明确了重要性和意义后，要围绕着目标再次确定为了成功需要探讨哪些问题。思考需要探讨什么问题可以帮助被教练对象围绕着目标来构建要解决的问题。

# 围绕预期成果展开谈话

教练没有在开始的时候弄清楚目标成果的真正实质，或者一心想要帮人解决问题这种做法会对教练过程形成阻碍作用。很多教练会出于良好的意图，却以不同的方式讲谈话带到偏离实质的地方。比如，被教练者想讨论如何提升影响力，教练如果觉得话题太大，就会将话题转为例如提升职业技能等方向。这种做法是是不可取的，教练需要听到了需要改变方向的可能性，并把他放给被教练对象思考，得到明确的回复后才能转换合约话题，而不是在没有达成一致的情况下直接改变方向。

教练要保持好奇，去了解目标对被教练对象有多大的意义，成功看起来，听起来，感觉上是什么样子？他们如何知道他们已经解决了这一问题或实现了目标？在合约阶段，要围绕着教练对象就如何成功展开话题。教练要与被教练者达成默契，成为真正的合作伙伴，在会谈过程中要经常性地核查，以确定议程和成果的标准释放能够完成，或者为了达到目标是否需要做出改变。

# 核心能力要点

  1. 确保询问了一个话题以后，可以了解清楚该话题的目标成果。
  2. 确保教练对象已经为会谈确定了衡量成功的标准。
  3. 帮助教练对象探讨话题中的意义和动力所在。
  4. 帮助教练对象确定为了实现目标成果需要解决什么问题。
  5. 牢记约定的目标成果，核查进展。
